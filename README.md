# HRMS Lite

A lightweight Human Resource Management System for internal admin use.

## Features
- Employee management (add, list, delete)
- Attendance tracking
- RESTful APIs
- Clean and professional UI

## Tech Stack
- Backend: Node.js, Express, Sequelize, SQLite
- Frontend: React (Vite)
- Deployment: Render & Netlify

## Getting Started

### Backend
```bash
cd backend
npm install
npm run dev

###Frontend
cd frontend
npm install
npm run dev
